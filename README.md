# 🛒 React Shopping Cart

A simple shopping cart built with React and useState. No libraries used.

## Features

- Add items to cart
- Remove items from cart
- Update quantity from cart table
- Live item count badge
- Live total price

## How to run

```bash
npm install
npm start
```

- useState hook to manage cart state
- Array methods like .map(), .filter(), .find()
- Passing functions as props
- Calculating totals with a for loop
